::: tip

如果你希望可以进行在线打印，而不是使用大SD 卡，或者因为找不到他的读卡器请使用树莓派进行打印

:::



## 基础环境

- 树莓派3B 及以上机型

  这个实验室有很多

- Octopi

  用于在线打印的镜像（请自行百度/Google下载）

  [下载](https://octoprint.org/download/)



当你准备好电源线及读卡器等设备时，请将Octopi的镜像烧录到树莓派的系统mircoSD卡中。(如果不会，请百度一下)

并按照配置修改其通过wifi连接（如果你不想使用网线连接的话）
